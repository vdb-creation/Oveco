## 🚀 Prochaines étapes

### 1. Finaliser l'installation WordPress
- Visitez : http://localhost/oveco/wp-admin/install.php
- Utilisez la base de données `oveco_local`

### 2. Activer le thème Oveco
- Allez dans **Apparence > Thèmes**
- Activez le thème **Oveco**

### 3. Commencer le développement
- Modifiez les templates dans `templates/`
- Personnalisez les styles dans `style.css`
- Ajoutez des fonctionnalités dans `inc/`
